🚀 Overview

Notes Pilot is a modern AI-powered study planner designed to help students organize their academic life efficiently.

It combines task management, study scheduling, notes organization, reminders, and productivity tools into one clean platform.

Whether you're preparing for exams, managing assignments, or building daily study habits, Notes Pilot helps you stay focused and productive.
